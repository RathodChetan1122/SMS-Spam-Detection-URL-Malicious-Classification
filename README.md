# 🔐 SMS Spam Detection & URL Malicious Classification

## 📌 Overview
This project focuses on using **Machine Learning** techniques to automatically detect **spam SMS messages** and **malicious URLs**. The system analyzes patterns in text messages and URLs to classify them as safe or harmful, helping reduce cyber fraud, phishing attacks, and unwanted spam.

## 🧠 Core Idea
The core idea of this project is to **automate the detection of spam messages and malicious URLs** using machine learning models trained on historical data, thereby improving digital security and reducing human effort.

---

## 🧩 Project Components

### 1️⃣ SMS Spam Detection
- Classifies SMS messages as **Spam** or **Ham (Legitimate)**
- Uses text preprocessing and feature extraction
- Applies machine learning classification models

### 2️⃣ URL Malicious Classification
- Classifies URLs as **Malicious** or **Benign**
- Learns patterns commonly found in phishing or harmful links
- Helps prevent users from accessing dangerous websites

---

## ⚙️ Machine Learning Workflow
1. Data collection
2. Data preprocessing (cleaning, tokenization, normalization)
3. Feature extraction
4. Model training
5. Model evaluation
6. Prediction on new SMS messages or URLs

## 🚀 Execution
The project is executed locally by running the Python scripts after installing the required dependencies.  
Models are trained on datasets and then used to predict whether an SMS or URL is safe or malicious.
